Este proyecto corresponde al ejercicio de CRUD de Películas utilizando LocalStorage, desarrollado con HTML, CSS y JavaScript.

La aplicación permite registrar películas, listarlas, editarlas y eliminarlas. Además, cuenta con un buscador por título y un filtro por género, lo que facilita la navegación y gestión de la información.  
Cada película puede verse de forma individual con sus detalles y es posible desplazarse entre ellas.

Toda la información se almacena en el LocalStorage del navegador, por lo que los datos se mantienen incluso al recargar la página.

Para ejecutar el proyecto basta con abrir el archivo index.html en el navegador o utilizar Live Server desde Visual Studio Code.

Tecnologías utilizadas:
HTML5, CSS3, JavaScript, Bootstrap y LocalStorage.
