# Red Social - React (scaffold)

Proyecto React basado en la plantilla `plantilla-RedSocial.html`.

Comandos:

```bash
cd hotel_app/redsocial-react
npm install
npm run dev
```

Siguientes pasos:
- Ajustar componentes y extraer partes repetidas.
- Mover assets locales (si tienes imágenes o CSS locales) a `public/`.
- Añadir enrutado si es necesario.
